# Bus Ticket Booking Application

A full-stack bus seat reservation system built with React, Node.js, and MongoDB.

## Live Demo
Frontend: https://bus-booking-appliaction.netlify.app

Backend API: https://bus-booking-backend-yni2.onrender.com

## Tech Stack

Frontend
- React
- CSS
- Vite

Backend
- Node.js
- Express
- MongoDB Atlas

Deployment
- Netlify (Frontend)
- Render (Backend)

## Features

- Bus seat reservation (40 seats)
- Passenger booking form
- Real-time seat availability
- Passenger dashboard
- Delete booking
- MongoDB database storage
